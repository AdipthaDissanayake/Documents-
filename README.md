# 🍔 Online Food Delivery Data Warehouse

## 📌 Project Overview
This project demonstrates the design and implementation of a Data Warehouse for an Online Food Delivery system. The goal is to transform operational data into a centralized repository that supports business intelligence and analytical reporting.

The project includes:

- Data warehouse architecture design
- ETL (Extract, Transform, Load) process
- Star schema dimensional modeling
- Fact and dimension tables
- SQL queries for analytical reporting
- Business-oriented insights

---

## 🎯 Objectives

- Integrate data from multiple sources
- Design an optimized dimensional model
- Build ETL pipelines for data transformation
- Support decision-making through analytical queries

---

## 🏗️ Data Warehouse Schema

### Fact Table
- FactOrders

### Dimension Tables
- DimCustomer
- DimRestaurant
- DimDriver
- DimDate
- DimLocation
- DimPayment
